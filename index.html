Como Publicar os PDFs

Para ativar o download dos livros:

1. Crie uma pasta chamada /livros/ na raiz do seu servidor/site.

2. Faça upload dos 8 arquivos PDF com os nomes exatos listados em cada card.

3. Os botões de download funcionarão automaticamente quando os arquivos estiverem no local correto.

Exemplo de estrutura:
/livros/
├── EU_SOU_SEU_DEUS.pdf
├── As_peças_da_minha_mente.pdf
├── A_QUEDA_DA_DOUTRINA.pdf
└── ...

Até lá, você pode ler o primeiro capítulo de cada obra e apoiar o projeto via Pix para acelerar a publicação completa.
